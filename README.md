# weIM

### 运行
1. 在终端执行 

    php websocket_server.php
    
2. 直接通过web服务器访问到 `public/index.php`。如果是服务器部署，请修改 `config/app.php` 中的 `server_url` 项


### 开发环境
> * PHP >= 5.4
> * A POSIX compatible operating system (Linux, OSX, BSD)
> * POSIX and PCNTL extensions for PHP
> * 使用workerman框架开发
